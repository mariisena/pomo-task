PomoTask ⏱️

Extensão para Chrome que ajuda você a gerenciar suas tarefas usando a Técnica Pomodoro! Foco, tempo e produtividade, tudo num clique.

🧪 Como instalar (modo desenvolvedor)

  1. Acesse chrome://extensions
  2. Ative o Modo do desenvolvedor (canto superior direito)
  3. Clique em Carregar sem compactação
  4. Escolha a pasta do projeto com o manifest.json
  5. Use o ícone da extensão para abrir o popup

🔐 Permissões usadas

  • storage: para salvar seus ciclos
  • tabs: (se aplicável, remover se não usar)
